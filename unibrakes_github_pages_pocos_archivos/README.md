# UniBrakes - GitHub Pages

Versión con pocos archivos para subir desde el navegador de GitHub.

## Sube estos 4 archivos

- index.html
- assets.js
- .nojekyll
- README.md

Esta versión evita el límite de GitHub de subir más de 100 archivos a la vez.

## Publicar

1. Crea un repositorio en GitHub.
2. Sube los 4 archivos.
3. Ve a Settings > Pages.
4. Selecciona Deploy from a branch.
5. Selecciona main y /root.
6. Guarda y espera la URL.
